A Java 21 / Spring Boot 3.x sandbox for building small, self-contained MVP demos
(backend prototypes to explore an idea or pattern). Not production code.

Bootstrapped from a reusable Claude Code template that wires in the `codegraph`
MCP server plus curated commands, skills, and agents (see `.claude/` and `.mcp.json`).

Build with Gradle: `./gradlew bootRun`, `./gradlew test`, `./gradlew build`.
